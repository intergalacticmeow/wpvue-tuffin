# WP Vue Theme: Tuffin

## Install

Copy `.env.example` to `.env` and configure the following settings/options:

* `VUE_APP_POSTS_PER_PAGE` - Defaults to `3` posts per/page

```shell
# npm run css
# npm run build
# npm run dev
```
